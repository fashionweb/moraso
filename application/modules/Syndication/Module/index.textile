h1. Syndication.Module

* Author: Andreas Kummer, w3concepts AG, (c) 2012

h2. Description

The Syndication.Module is somehow special in the sense, that it is not used directly, but as a dynamic substitution of parent modules (such as List.Download).
If a module is configured as a parent of Syndication.Module in the hierarchy.ini (/application/modules/hierarchy.ini) the user may substitute the current module with Syndication.Module. The syndication module then allows to specify an article to take the content from, fetches the content of the same module with the same index from the specified page.

h2. Use

As mentioned above the module may not be used directly. It can be specified as a descendant of a module that extends Aitsu_Module_Tree_Abstract either in the /application/modules/hierarchy.ini or in the .

h2. Static invocation (without editable configuration)

If you intend to use the same content over and over again, you should not use this module. In this case you may put following script shortcode into the template:

bc. <script type="application/x-aitsu" src="My.Module:1">
	edit = false
	execcontext = 123
</script>

# Replace My.Module:1 with the reference to the module of your choice;
# Replace 123 (execcontext) with the idartlang of the source article.
